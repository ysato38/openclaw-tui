# OpenClaw-TUI Landing

## 開発
```bash
npm install
npm run dev
```

## ビルド
```bash
npm run build
npm run start
```

## デプロイ（Vercel）
1. Vercelに `landing` ディレクトリをRoot Directoryとして連携
2. Build Command: `npm run build`
3. Output: Next.js標準

## 対応要件
- レスポンシブ（SP/Tablet/Desktop）
- Tailwind + Framer Motion
- SEO / OGP / JSON-LD
- CTAのコピーボタン
- Footer + GitHub/License
